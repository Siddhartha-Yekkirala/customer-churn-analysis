# Customer Churn Analysis 📊

## 📌 Project Overview
This project analyzes customer churn data to identify patterns and key factors that lead to customer attrition. The goal is to help businesses improve customer retention strategies.

---

## 🛠️ Tech Stack
- Python (Pandas, Matplotlib)
- SQL
- Tableau

---

## 📊 Key Insights
- Customers with higher monthly charges have higher churn rates
- Month-to-month contract users are more likely to churn
- Low tenure customers show higher churn probability

---

## 📈 KPIs
- Churn Rate
- Customer Segmentation
- Revenue Impact

---

## 📂 Project Structure
- data/ → Dataset
- notebooks/ → Python analysis
- sql/ → SQL queries
- dashboard/ → Tableau dashboard
